# Search Console and Indexing Playbook

## Primary sitemap index
- Submit: `https://convertfiles.qzz.io/sitemap.xml`

## Segmented sitemap files
- `https://convertfiles.qzz.io/main-sitemap.xml`
- `https://convertfiles.qzz.io/tools-sitemap.xml`
- `https://convertfiles.qzz.io/guides-sitemap.xml`
- `https://convertfiles.qzz.io/compare-sitemap.xml`
- `https://convertfiles.qzz.io/categories-sitemap.xml`
- `https://convertfiles.qzz.io/bangla-sitemap.xml`

## Google Search Console checklist
1. Verify the `convertfiles.qzz.io` property.
2. Submit the main sitemap index.
3. Use URL Inspection on the homepage and the top 15 priority URLs.
4. Request indexing for new guides, compare pages, and category hubs.
5. Review coverage, page experience, and enhancements weekly.

## Bing Webmaster Tools checklist
1. Verify the site property.
2. Submit the same sitemap index.
3. Request indexing for the new high-priority URLs.
4. Configure IndexNow if you want faster Bing discovery.

## Suggested GA4 report segments
- landing page path
- source / medium
- country
- device category
- event name
- page title

## Recommended custom event review
- `tool_viewed`
- `tool_started`
- `tool_completed`
- `next_tool_click`
- `share_click`
- `search_click_landing`
- `organic_guide_cta_click`

## Practical note
Indexing speed is not guaranteed. Expect Search Console data to lag, and use week-over-week movement rather than day-one expectations.
